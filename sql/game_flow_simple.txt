================================================================================
                    KEY PATHS TO ENDINGS
================================================================================

PATH 1: Ending1 - Hero's Return (via Amulet)
  intro → ruins → Examine pedestal → ruins (get amulet) → temple → 
  Use amulet → Enter temple → ending1 ✓

PATH 2: Ending1 - Hero's Return (via Key from Cave)
  intro → forest → Search supplies → forest (get torch) → cave → 
  Descend deeper (needs torch, get key) → treasure → Take key/amulet → 
  forest → Follow path to village → village → Continue to temple → 
  temple → Use key → Enter temple → ending1 ✓

PATH 3: Ending1 - Hero's Return (via Key from Dungeon)
  intro → forest → Follow footprints → cave → Explore side passage → 
  dungeon → Search for weapon (get sword) → Fight creature (get key) → 
  treasure → Take key/amulet → forest → Follow path to village → 
  village → Continue to temple → temple → Use key → Enter temple → ending1 ✓

PATH 4: Ending1 - Hero's Return (via Force)
  intro → (get sword from dungeon/treasure) → temple → 
  Force door open (sword, -25 HP) → Enter temple → ending1 ✓

PATH 5: Ending2 - Power Within
  intro → ruins → Read symbols → ruins (get knowledge) → temple → 
  Study symbols → Absorb power → ending2 ✓

PATH 6: Death (Health Loss)
  intro → (any path with health loss) → (health <= 0) → death ✗

================================================================================
                    IMPORTANT NOTES
================================================================================

• Loops are not shown in this flowchart (self-loops like "Search supplies" 
  that return to the same scene)

• Health loss can occur in:
  - danger: -10, -30, -50 HP
  - danger2: -5, -15, -20 HP
  - bridge: -15 HP (risky crossing)
  - temple: -25 HP (force door)

• Health restoration available at:
  - heal: +25 HP or +100 HP

• Terminal scenes (game ends):
  - ending1: Hero's Return
  - ending2: Power Within  
  - death: Game Over

• Items needed for endings:
  - ending1: amulet OR key OR sword
  - ending2: knowledge
